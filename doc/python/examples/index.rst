Examples
--------

Thesescripts illustrate the use of the FreeDiscovery Python
package on a subset of the TREC 2009 legal collection. See the README for the
quick start instructions that use `curl`. 

These examples can be run with e.g.,

   .. code-block:: bash

       python examples/python/categorization_interpretation.py
