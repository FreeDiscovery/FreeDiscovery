Python package
==============

Installation
------------

FreeDiscovery Python package requires **Python 3.5+**, as well as,

 * numpy >= 1.10.1
 * scipy >= 0.16.1
 * pandas >= 0.18.0
 * scikit-learn >= 0.18.0

It can be installed with,

.. code:: bash

    pip install freediscovery
